## Orientation

1. **Drawing "flat" /2.5d**
   1. Control + N
   2. Go to Edit Mode
2. **Saving**
   1. **Save project:** File  > Save as&#x20;
   2. **Save snapshots:** Document  > Save Document as
3. **Make polymesh3D**
4. **Dynamesh (tools)**
5. **Materials**
   1. **MatCap Gray - Better color**

```javascript
//Shortcuts
//Brushes
Invert brush     Alt LMB    
Draw key    S            
Brush menu    B        
    Standard    BST        
    Clay        BCT       
    Dam         BDS
    Planar      BDT       
    Move        BMV       
    Smooth      Hold Shift
    Trim        Shift + ctrl

// View
Wireframe    Shift + F
Subdivide    Cmd + D
Symmetry     x


//General

Cmd + N     New project
Cmd + Q     Quit
```
