# InDesign

## Core skills

* paragraph and character style&#x20;
* margins and columns&#x20;
* placing images

## Shortcuts

```
hide guide - w 
preview mode - shift + w
exit preview - esc 
show ruler - cmd + r
hide guides - cmd + " ?

text panel - command + t
```

## Design principles

$$
alignment  \times consistency  \times legibility  \times contrast
